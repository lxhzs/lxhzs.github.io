    • line-through 删除线
    • none 清除默认下划线

a标签的伪类

  • a:visited{属性:值;}
  • a:hover{属性:值;}
  • a:active{属性:值;}
  • a:focus{属性:值;}

3.块级元素和行内元素水平居中的方式

  • margin: 0 auto 块元素居中对齐
  • text-align:center让块元素和行内块元素里边的内容水平居中

4.border 边框线型

  • solid; 实线
  • dashed; 虚线

7.表格边框合并

8.表单input的默认样式

  • 去掉边框
  • 去掉轮廓线

9.浮动 float

  • 浮动的特点
    • 浮动元素会脱离标准流，不占据原来的位置。
    • 浮动元素会在一行上显示。
    • 浮动元素，即使代码换行，中间没有空隙。
    • 浮动元素，不设置宽高，默认宽高为0，宽高是内容撑开的宽高。
    • 浮动元素只影响后边的元素，不影响前边的元素。
    • 浮动元素顶对齐。
    • 浮动可以将行内元素转换成行内块元素显示模式(不推荐使用，转行内块推荐使用display:inline-block;)

10.文本饶图

  • 文本不参与浮动
  • 图片给他一个浮动

11.清除浮动

  • 给父元素设置高度
  • 额外便签法
    • 给最后一个浮动元素之后，添加额外标签，给这个标签使用clear:both;

  • 给浮动元素的父元素使用overflow:hidden; 触发bfc

  • 单伪元素清除浮动

    • 给浮动元素父元素使用伪元素清除浮动
  • 双伪元素清除浮动

12.定位 position

  • 静态定位 static

  • 绝对定位 absolute

    • 绝对定位的元素会脱离标准流，不占据原来的位置
    • 绝对定位的元素，位置基于浏览器偏移。
    • 嵌套的盒子，父元素没有使用定位，子元素使用绝对定位的位置基于浏览器偏移。
    • 嵌套的盒子，父元素使用绝对定位，子元素使用绝对定位，子元素位置基于父元素的位置偏移。
    • 行内元素使用绝对定位之后，显示模式转换成了行内块(不推荐使用，转行内块推荐使用display:inline-block;)

  • 相对定位 relative

    • 元素使用相对定位，不脱离标准流，还占据原来的位置。
    • 元素使用相对定位，位置基于自身的位置偏移。
    • 子绝父相，子元素绝对定位，父元素相对定位，子元素的位置基于父元素的位置偏移。
    • 相对定位不能实现行内元素转行内块元素。

  • 固定定位 fixed

    • 元素使用固定定位，位置基于浏览器偏移。
    • 元素使用固定定位，脱离了标准流，不占据原来的位置。
    • 可以将行内元素转换为行内块显示模式(不推荐使用，转行内块元素，推荐使用display:inline-block;)

13.定位的盒子水平居中显示

  • margin: 0 auto; 只能让标准流的块元素居中显示。

  • 首先使用left:50%;向走父盒子的一半，再是margin-left负值，向左走自身的一半，实现定位的盒子水平居中显示。

  • css3里的transform translate属性

14.vertical-align 属性

  • baseline; 基线对齐 默认值
  • top; 顶线对齐
  • middle;中线对齐
  • bottom;底线
  • 对行内块元素敏感。行内块的元素对齐，可以使用vertical-align

15.Overflow 溢出处理

  • visible; 溢出显示 默认值
  • hidden; 溢出隐藏
  • auto; 内容溢出生成滚动条，不溢出不生成滚动条
  • scroll; 内容是否溢出都生成滚动条

16.透明度问题

  • Opacity: 不透明度 范围0-1 改变元素所有内容的不透明度
  • filter:alpha(opacity=30); 兼容ie6~9的不透明度 opactiy范围从0-100
