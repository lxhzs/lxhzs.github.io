Js & JQuery 插件整理

1. 触屏滚动

  • fullPage 响应式全屏滑动插件
  • iSlider 响应式全屏滑动插件
  • superslide2 PC端网站的特效插件
  • TouchSlide 一款移动端触屏滑动特效插件by大话主席
  • swiper 常用于移动端网站的内容触摸滑动
  • zepto.fullpage 专注于移动端的fullPage.js

2. 视差滚动

随着滚动条的滚动，页面上的元素像动画一样，一帧帧展示出来。

  • SUPERSCROLLORAMA 国外的一款插件
  • jQuery全屏滚动插件fullPage.js fullPage.js的一些使用demo demo

3. 日期选择器

  • bootstrap-datetimepicker 此项目是bootstrap-datetimepicker项目的一个分支，进行了改进、增强
  • jq-timeTo 国外的一款时钟插件，时间会动态切换
  • flipclockjs 国外的一款时钟插件，时间会百叶窗式的切换

4. 图片

  • baguetteBox.js - 是一个简单易用的响应式图像灯箱效果脚本。demo
  • Lightgallery.js - 是一个功能齐全的JavaScript图像灯箱插件。demo
  • viewerjs - 是一个图像预览插件。demo
  • cropperjs - 是一个图片编辑器插件。demo
  • blazy.js - 是一个懒惰加载插件。demo
  • PhotoSwipe - 支持各种手势操作的灯箱效果 demo

5. 布局

  • SuperEmbed.js - 是一个Javascript库，可检测出网页上的内嵌视频并使他们能够变成响应式元素。demo
  • ScrollReveal - ScrollReveal一款瀑布流效果插件，上下滚动都会有懒加载效果。demo
  • Bricks.js -一款瀑布流布局插件。demo

6. 弹出层

  • animatedModal - 一款英文插件，全屏弹出框
  • layer - 功能强大的弹出层插件
  • artDialog - 国内的一款插件，文档很详细 demo
  • Popper.js - 是一个轻量级的库用于管理工具提示和弹窗效果。demo

7. 音频视频

  • Loud Links - 是一个轻量级的JavaScript库用于添加交互声音到您的站点。demo
  • flv.js - B站HTML5播放器内核开源。

8. 图表

  • echarts - 一个使用JavaScript实现的开源可视化库 demo
  • hcharts - 兼容IE6+、完美支持移动端 demo
  • chart - 一款灵活的JavaScript图表工具

9. 幻灯/切换

  • pageSwitch - 适用场景为全屏切换，即一切一屏，并且有百种切换效果 demo
  • Swiper - 移动端轮播图滑动插件（API英文） demo
  • H5-fullscreenpage.js - 一款轻量级的移动端全屏切换插件

10. 动画

  • jquery.transit - 一款JQuery插件，弥补$.fn.animate不能实现的动画效果
  • animate.css - 一款CSS动画插件
  • anime.js - 是一个灵活轻便的JavaScript动画库。demo
  • three.js - 是一个JS 3D库。demo
  • vivus - JavaScript库，使SVG绘制动画。demo

11. 表单

  • validator.js - 轻量级的JavaScript表单验证，字符串验证，中文文档。demo
  • List.js - 是一个轻量级的为列表、表格或其他任何HTML标签增加了搜索，排序，过滤器和灵活性等元素，英文文档。demo
  • validator.js - 是一个简单、轻量级，但功能强大的Validator 组件。demo

12. 存储

  • store.js - 本地存储localstorage的封装，提供简单的API。demo
  • cookie.js - 对操作cookie的封装，提供简单的API 兼容IE6。demo

13. 常用插件网站

  • plugins.jquery - jQuery的插件库
  • 开源中国jQuery插件库
  • dowebok - 素材demo库
  • jQuery插件库 -收集最全最新最好的jQuery插件
  • jQuery之家 - 提供了大量的jQuery插件
  • HTML5资源库 - 收藏html5资源教程最完整的html5学习网站
  • 前端组件库大合集-必备收藏

14. 其他

  • 上拉加载更多 - 依赖zepto或者jQuery的一款移动端加载插件
  • html5shiv - 解决低版本浏览器不支持HTML5
  • 在线代码片段 - 集合各种语言的在线代码片段
